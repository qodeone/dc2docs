Code Recipes
============

A list of code samples/examples outlining how to create and load
commerce entities entirely in code.

These recipes are all designed to work off of each other, so try it out
- you can run all the code in a single shot from top to bottom.

`Stores <stores.rst>`__ - Stores and types.

`Variations <variations.rst>`__ - Product variations and types.

`Attributes <attributes.rst>`__ - Product attributes and values.

`Products <products.rst>`__ - Products and types.

`Orders <orders.rst>`__ - Orders, order items, and their types.
